# ClusPy

Clustering data in python.

## Implemented algorithms

- Density-based
    - Multi Density DBSCAN
- Centroid-based
    - X-Means
    - G-Means
    - PG-Means
    - Fuzzy-C-Means
- Non-redundant clustering (alternative clustering)
    - NR-Kmeans
- Subspace clustering
    - SubKmeans
    - MDL-SubKmeans
